## Running locally

Open this project via a local web server (not `file://`), otherwise the Firebase ES module imports may be blocked and you’ll see a blank page.

- **VS Code**: use the “Live Server” extension and open `index.html`
- **Node** (if installed):

```bash
npx --yes serve .
```
